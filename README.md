## About
- 기술 면접을 위한 **개발 관련 지식**을 정리하는 저장소입니다.
- 매주 정한 주제에 대해 각자 **스터디 자료**들을 커밋합니다.
- 학습 범위에 대하여 모의면접을 진행하며 **질의응답** 형태로 문서화하는 것을 목표로 합니다.
